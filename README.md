# 我的学习笔记

> 持续积累的知识库，记录我在技术、学习和思考中的点滴收获。

## 📖 关于这个仓库

这是一个公开的个人学习笔记仓库，使用 **Obsidian** 管理和维护。这里记录了我日常学习中遇到的知识点、问题解决方案、技术心得等内容。

将笔记公开的目的是：
- **知识沉淀**：通过整理和分享，加深对知识的理解
- **交流学习**：欢迎讨论和指正，共同进步
- **方便回顾**：随时随地可以查阅和复习

## 📚 内容分类

- **AI 学习** - 人工智能相关笔记
- **考研相关** - 考研英语、数学等复习资料
- **技术文档** - 编程、工具使用等技术笔记
- **方案记录** - 各种技术方案的实现记录

## 🛠️ 技术栈

- **笔记工具**：[Obsidian](https://obsidian.md/)
- **主题风格**：使用 Markdown 格式，支持 Wiki 链接、嵌入、标注等

## 📝 如何使用

如果你使用 Obsidian，可以直接克隆这个仓库作为你的 Vault：

```bash
git clone https://github.com/你的用户名/仓库名.git
```

然后在 Obsidian 中打开该文件夹即可。

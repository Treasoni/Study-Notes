**一句话先记住**

>- **宽带**：你办的“网络套餐 / 上网通道”
>- **带宽**：这条通道**有多宽、一次能跑多少数据**

# 1.  宽带是什么？

👉 **宽带 = 一条高速公路**

>- 你家装的 **100M / 300M / 1000M 宽带**
>- 移动 / 电信 / 联通给你拉的一根网线
>- 本质：**能上网这件事本身**

> 没有宽带，就等于 **没路**，车根本跑不了。


# 2.  带宽是什么？

👉 **带宽 = 高速公路有几条车道**

>- 带宽越大 = 车道越多
>- 同一时间能跑的数据就越多
>- 下载更快、看视频更流畅、多设备不卡

> 带宽小 = 单车道  
> 带宽大 = 八车道高速

# 3. 很多人容易混淆的一点（重点‼️）

## 3.1 你办的「100M宽带」

👉 **M = Mbps（兆比特）**

但你电脑里显示的下载速度是：  
👉 **MB/s（兆字节）**

## 3.2 换算关系

`1 MB/s ≈ 8 Mbps`

所以：

>- 100M 宽带
>- 理论下载速度 ≈ **12.5 MB/s**

> 看到只有 10~12 MB/s 是**正常的**，不是被坑了

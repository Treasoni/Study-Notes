# 1. 创建虚拟机，本质在做哪些事？

先给你一句**总纲**：

> **创建虚拟机 = 给一台“假电脑”分配硬件资源 + 接好网络 + 准备装系统**

这台“假电脑”会有：

|真实电脑|虚拟机|
|---|---|
|主板|虚拟主板|
|CPU|vCPU|
|内存|虚拟内存|
|硬盘|虚拟磁盘|
|网卡|虚拟网卡|
|U 盘 / 光驱|ISO|

## 2.2 OS（操作系统）

 **核心问题：**

👉 **这台虚拟机是“怎么启动的”？**

### 2.2.1 常见选择

**✔ Use CD/DVD disc image file (iso)**

- 用 ISO 装系统（Linux / Windows）

#### ✔ Do not use any media

- iStoreOS / OpenWRT 常用
- 稍后手动导入磁盘

### ISO 文件从哪来？

`Datacenter → Storage → local → ISO Images`

# 写盘工具对比

> [!info] 概述
> 写盘工具用于将系统镜像（ISO）写入 U 盘制作启动盘。
> 类比：就像把系统安装盘"复制"到 U 盘里，让电脑可以从 U 盘启动安装系统。

## 核心概念 💡
- **写盘**：将 ISO 镜像文件以低层方式写入 U 盘，制作成可启动的安装盘
- **分区表**：MBR（Legacy BIOS）vs GPT（UEFI），影响启动兼容性
- **文件系统**：FAT32、NTFS、exFAT，决定 U 盘的容量和兼容性

## 主流写盘工具对比

### Rufus（最常用，成功率最高）

> [!tip] 特点
> - 真正的"低层写盘"，完全重写 U 盘数据
> - 支持 BIOS/UEFI 双模式启动
> - 可选 MBR/GPT 分区表格式
> - 可选 FAT32/NTFS 文件系统
> - 可绕过 Windows 11 TPM/Secure Boot 限制

> [!success] 适用场景
> - Windows 安装盘（特别是 Win11）
> - Linux 发行版启动盘
> - 对启动兼容性要求高的场景

| 优点 | 缺点 |
|------|------|
| 成功率极高 | 新手看到选项容易不知所措 |
| 细节完全可控 | 界面相对简陋 |
| 支持绕过 Win11 限制 | |

**一句话评价**：成功率第一，推荐首选

### balenaEtcher（最"无脑"）

> [!tip] 特点
> - 三步走流程：选镜像 → 选 U 盘 → 写入
> - 不让用户选择文件系统、分区表
> - 严格的写入验证，写坏了直接报错
> - 跨平台支持（Win/macOS/Linux）

> [!success] 适用场景
> - Linux 镜像写入
> - 树莓派、嵌入式系统镜像
> - 新手、懒人用户

| 优点 | 缺点 |
|------|------|
| 操作简单，不会出错 | 几乎没有自定义选项 |
| 不会写错盘（有验证） | 不支持 Windows 特殊需求 |
| 界面现代美观 | |

**一句话评价**：懒人/新手/Linux 镜像首选

### Ventoy（最灵活）

> [!tip] 特点
> - 一次制作，多次使用
> - 制作成 Ventoy 启动盘后，直接拷贝 ISO 文件即可
> - 启动时选择要进入哪个系统
> - 支持 Win + Linux + PE 多系统共存

> [!success] 适用场景
> - 一个 U 盘放多个系统
> - 经常更换 ISO 镜像
> - 装机党、折腾党、系统管理员

| 优点 | 缺点 |
|------|------|
| 不用反复写盘 | 启动链稍复杂 |
| 可以放很多个 ISO | 个别老主板兼容性一般 |
| 支持 ISO/WIM/VHD/IMG | |

**一句话评价**：折腾玩家神器

## 操作步骤

### 使用 Rufus 制作启动盘

1. 下载 Rufus：https://rufus.ie（最新版本）
2. 插入 U 盘（**会清空所有数据**，注意备份）
3. 选择 ISO 镜像
4. 设置分区类型（MBR/GPT）和目标系统类型
5. 点击开始，等待写入完成

### 使用 balenaEtcher 制作启动盘

1. 下载 Etcher：https://www.balena.io/etcher（最新版本）
2. 打开软件，选择 ISO 镜像
3. 选择目标 U 盘
4. 点击 Flash，等待写入和验证完成

### 使用 Ventoy 制作多系统启动盘

1. 下载 Ventoy：https://www.ventoy.net（最新版本）
2. 运行 Ventoy，选择 U 盘安装
3. 安装完成后，直接将 ISO 文件拷贝到 U 盘
4. 启动时从 Ventoy 菜单选择系统

## 注意事项 ⚠️

1. **写盘会清空 U 盘所有数据**，操作前确认 U 盘没有重要文件
2. **Rufus 的分区类型选择**：
   - 老电脑/BIOS 启动：选 MBR
   - 新电脑/UEFI 启动：选 GPT
3. **Windows 11 绕过限制**：在 Rufus 的镜像选项里可以绕过 TPM 和 Secure Boot 检查
4. **Ventoy 的 ISO 格式**：直接拷贝 ISO 文件，不要解压或修改

## 常见问题 ❓

**Q: 写盘后 U 盘容量变小了？**
A: 这是正常的，写盘工具会重新分区。用资源管理器格式化无法恢复，需要用写盘工具或磁盘工具重新格式化。

**Q: Rufus 显示"设备被占用"？**
A: 关闭可能访问 U 盘的程序（资源管理器、杀毒软件等），或者重新插拔 U 盘。

**Q: Ventoy 启动蓝屏？**
A: 个别老主板与 Ventoy 兼容性问题，可以尝试在 Ventoy 设置里启用兼容模式，或者换用 Rufus。

**Q: 哪个工具最推荐？**
A: 通用场景推荐 Rufus，新手推荐 Etcher，需要多系统用 Ventoy。

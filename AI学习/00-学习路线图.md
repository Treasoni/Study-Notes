# AI 学习路线图

> [!tip] 学习原则
> **从易到难，从实用到高级** - 每一步都能立即在实践中使用

## 学习路径概览

```
第一阶段：基础概念
    ↓
第二阶段：实践工具
    ↓
第三阶段：进阶能力
    ↓
第四阶段：高级应用
```

---

## 第一阶段：基础概念

### 1. Prompt（提示词）
**文档**：[[Prompt, Agent, MCP 是什么]]

**学习要点**：
- System Prompt vs User Prompt
- 角色设定的重要性
- 提示词编写技巧

**实践**：
- 写一个让 AI 扮演特定角色的提示词
- 体验不同 System Prompt 的效果差异

---

### 2. Agent 和 Function Calling
**文档**：[[Prompt, Agent, MCP 是什么]]

**学习要点**：
- Agent 的工作原理
- Function Calling 的标准化作用
- Tools 和 Tool Registry

**实践**：
- 理解 Agent 如何调用工具
- 了解 JSON Schema 的作用

---

## 第二阶段：实践工具 ⭐ 重点

### 3. Claude Code 基础使用
**文档**：[[Claude Code 常用功能]]、[[如何使用Claude code]]

**学习要点**：
- 安装和配置
- 基本命令和 Slash 命令
- 文件操作和搜索
- 多平台配置

**实践**：
- 安装 Claude Code
- 配置一个兼容平台（如 DeepSeek）
- 使用基本命令操作文件

---

## 第三阶段：进阶能力 ⭐ 重点

### 4. Skills 编写
**文档**：[[Skills 是什么]]、[[如何编写Skills]]

**学习要点**：
- Skills 的核心概念
- metadata.json 编写
- skill.md 结构
- Skills 最佳实践

**实践**：
- 创建第一个简单 Skill
- 创建一个领域相关的 Skill
- 测试和优化 Skill

---

### 5. MCP 配置
**文档**：[[Claude MCP 使用指南]]

**学习要点**：
- MCP 协议原理
- stdio/SSE/HTTP/WebSocket 类型
- 常用 MCP 服务器
- 配置文件管理

**实践**：
- 配置 filesystem MCP
- 配置一个云端服务（如 GitHub）
- 测试 MCP 连接

---

## 第四阶段：高级应用

### 6. Subagent 使用
**文档**：[[Claude Subagent 使用指南]]

**学习要点**：
- 内置 subagent 类型
- 自定义 Agent 创建
- Agent 最佳实践

**实践**：
- 使用 Explore agent 探索代码库
- 创建一个自定义 Agent

## 快速参考

### 学习优先级

| 优先级 | 主题 | 时间投入 | 实用性 |
|--------|------|----------|--------|
| **高** | Claude Code 使用 | 1-2 天 | ⭐⭐⭐⭐⭐ |
| **高** | Skills 编写 | 2-3 天 | ⭐⭐⭐⭐⭐ |
| **中** | MCP 配置 | 1-2 天 | ⭐⭐⭐⭐ |
| **中** | Prompt 基础 | 1 天 | ⭐⭐⭐ |
| **低** | Agent 原理 | 1 天 | ⭐⭐ |
| **低** | Subagent | 1-2 天 | ⭐⭐⭐ |

### 学习建议

> [!info] 实用至上
> - **理论 + 实践结合** - 每学一个概念就动手实践
> - **从简单开始** - 先掌握基础，再深入高级
> - **记录笔记** - 将学习过程中遇到的问题记录下来
> - **循序渐进** - 不要试图一次性学完所有内容

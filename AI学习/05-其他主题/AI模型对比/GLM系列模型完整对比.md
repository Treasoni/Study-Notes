# GLM系列模型完整对比

> [!info] 概述
> **一句话定义**：GLM 是智谱AI（Z.ai）开发的大语言模型系列，从基础文本理解到多模态能力，覆盖从个人到企业的全场景需求。
>
> **🎯 比喻**：GLM 系列就像一个「AI产品家族」，从经济型轿车到豪华跑车，每种车型都有其最适合的使用场景。

## 核心概览

### GLM系列发展历程

```
GLM-4 (基础版) ─┐
                ├─→ GLM-4.5 (全能均衡型) ─┐
GLM-4V (视觉版)─┤                        ├─→ GLM-4.7 (高级开源)
                └─→ GLM-4.5V (视觉专家) ─┤
                                         └─→ GLM-5 (旗舰王者)
                                         └─→ GLM-4.5-Air (轻量快手)
```

### 各模型定位

| 模型 | 定位 | 核心优势 | 目标用户 |
|------|------|----------|----------|
| **GLM-5** | 旗舰王者 | 复杂系统、长期规划能力 | 企业级复杂项目 |
| **GLM-4.7** | 高级开源 | 综合性能、透明可审计 | 开发者、研究机构 |
| **GLM-4.5** | 全能均衡 | 推理、编码、智能体 | 通用开发场景 |
| **GLM-4.5V** | 视觉专家 | 图像、视频、文档理解 | 多模态应用 |
| **GLM-4.5-Air** | 轻量快手 | 快速响应、高性价比 | 个人用户、快速原型 |

### 分模型详解

#### 1. GLM-5 - 旗舰王者

```
┌─────────────────────────────────────────────────────────┐
│                      GLM-5                              │
├─────────────────────────────────────────────────────────┤
│ 参数：744B 总参数，40B 激活参数                          │
│ 上下文：200K+ tokens                                    │
│                                                         │
│ 核心技术：                                               │
│ ✓ DeepSeek Sparse Attention (DSA) - 降低部署成本        │
│ ✓ 异步 RL 训练框架 (slime) - 更精细的后训练迭代         │
│ ✓ 预训练数据：28.5T tokens (GLM-4.5: 23T)              │
│                                                         │
│ 适用场景：                                               │
│ • 复杂业务流程设计与执行                                 │
│ • 长期项目规划（Vending Bench 2: $4,432 收益）          │
│ • 企业级应用开发（直接生成 .docx/.xlsx/.pdf）            │
│ • 多步骤任务协作（Terminal-Bench: 56.2/60.7）           │
│ • 系统 Agent（BrowseComp: 62.0, τ²-Bench: 89.7）       │
│                                                         │
│ 硬件要求：                                               │
│ • 推荐配置：A100/H100 x4                                │
│ • 本地部署支持：vLLM、SGLang                             │
│ • 非 NVIDIA 芯片支持（华为、寒武纪等）                   │
└─────────────────────────────────────────────────────────┘
```

**特点详解**：
- **专精复杂系统工程**：从设计到执行的全流程能力
- **长期规划能力**：在 Vending Bench 2（1年模拟经营）中排名开源第一
- **文档生成能力**：直接输出可用的 Word、Excel、PDF 文件
- **Agent 模式**：Z.ai 官方应用支持，内置文档处理技能

### 定价策略（概览）

| 模型 | 定位 | 价格级别 | 免费额度 | 性价比 |
|------|------|----------|----------|--------|
| GLM-5 | 旗舰 | 高 | 有限 | 企业级 |

### 成本优化建议

1. **分级使用策略**
   ```
   简单任务 → GLM-4.5-Air（最省成本）
   日常开发 → GLM-4.5（性能价格平衡）
   复杂系统 → GLM-5（一步到位）
   ```

2. **批处理优化**
   - 合并多个小请求
   - 减少API调用次数
   - 利用完整的上下文窗口

3. **缓存策略**
   - 缓存常见问题回复
   - 避免重复计算
   - 使用向量数据库存储

4. **本地部署方案**
   - GLM-4.5-Air 适合本地部署
   - 一次性硬件投入，长期降低成本
   - 适合有数据隐私要求的场景

需要开源/透明度          → GLM-4.7
日常开发/通用场景        → GLM-4.5
个人使用/成本敏感        → GLM-4.5-Air
快速响应/实时对话        → GLM-4.5-Air
代码审查/重构            → GLM-4.5 或 GLM-4.7
文档生成 (.docx/.xlsx)   → GLM-5
```

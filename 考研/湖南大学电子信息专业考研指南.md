## 六、备考建议

### 6.1 目标分数设定

```
建议目标分数：360-370分以上

分项目标：
├── 政治：70+
├── 英语：70+
├── 数学：120+
└── 专业课（863）：110+
```

**理由**：
- 2025年物理与微电子学院复试线已达350分
- 各学院不接收调剂
- 分数线呈上升趋势

### 6.2 各科目复习策略

#### 政治（目标70+）
- 时间分配：每天1-2小时
- 重点：马原、毛中特、史纲、思修
- 资料：肖秀荣系列、徐涛强化班

#### 英语（目标70+）
- 时间分配：每天2-3小时
- 重点：阅读理解、作文
- 资料：真题、单词书、作文模板

#### 数学（目标120+）
- 时间分配：每天3-4小时
- 重点：高数、线代、概率论
- 资料：张宇、汤家凤、李永乐系列

#### 专业课863（目标110+）
- 时间分配：每天2-3小时
- 重点：模电、数电基础知识
- 资料：指定教材、历年真题

### 6.3 复试准备要点

1. **专业课笔试**：重点复习电子器件相关知识
2. **综合面试**：准备中英文自我介绍、专业问题回答
3. **英语口语**：日常练习，准备常见问题
4. **机试**（信科院）：提前练习编程能力，熟练C/C++/Python

### 6.4 时间规划建议

```
基础阶段（现在-6月）
├── 数学：一轮基础复习
├── 英语：背单词、学语法
└── 专业课：通读教材

强化阶段（7-9月）
├── 数学：刷题强化
├── 英语：阅读训练
├── 政治：开始复习
└── 专业课：重点章节突破

冲刺阶段（10-12月）
├── 全科：真题模拟
├── 政治：背诵押题
└── 专业课：查漏补缺
```

## 八、重要提醒

### 8.1 招生政策

1. **不接收调剂**：各学院电子信息专业均不接收调剂
2. **一志愿重要**：必须一次成功，没有调剂机会
3. **复试线可能高于学校线**：特别是热门学院和专业

### 8.2 复试注意事项

1. **机试要求**（信科院）：机试不合格者不予录取
2. **面试规格化**：综合面试可能采取规格化计算方法
3. **复试不及格线**：复试成绩低于144分不予录取

### 8.3 其他注意事项

1. 招生政策每年可能调整，关注官网最新通知
2. 复试通常在3月下旬进行
3. 复试费120元/人
4. 体检在入学报到后进行

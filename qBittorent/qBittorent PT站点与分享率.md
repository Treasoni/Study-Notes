# qBittorrent PT站点与分享率管理

## 1. 公站 vs 私站（PT）

### 1.1 公共站点（公站）

**特点：**

- 免费注册，无需邀请
- 不强制要求做种
- 资源质量参差不齐
- 速度不稳定

**示例：**
- 影视天堂
- BT之家
- HDChina（部分公共）

### 1.2 私人站点（PT - Private Tracker）

**特点：**

- 🔒 **需要邀请码注册**
- 📊 **强制考核分享率**
- ⚡ **资源质量高**（有人压制/整理）
- 🛡️ **社区氛围好**
- ⚖️ **严格的规则**

**示例：**
- M-Team（馒头）
- HDSky
- CHDBits
- TTG
- OurBits
- KeepFrds

## 2. 分享率（Ratio）详解

### 2.1 什么是分享率？

```
分享率 = 上传量 / 下载量
```

**举例：**

| 下载 | 上传 | 分享率 |
|------|------|--------|
| 10GB | 10GB | 1.0 |
| 10GB | 20GB | 2.0 |
| 10GB | 5GB | 0.5 |

### 2.2 分享率的重要性

| 分享率 | 状态 | 后果 |
|--------|------|------|
| ≥ 2.0 | 优秀 | 无限下载 |
| 1.0 - 2.0 | 良好 | 正常使用 |
| 0.5 - 1.0 | 警告 | 可能被限速 |
| < 0.5 | 危险 | 账号可能被封 |

### 2.3 常见站点规则

不同站点有不同的要求：

| 站点 | 最低分享率 | 考核周期 |
|------|------------|----------|
| M-Team | ≥ 1.0 | 每月 |
| HDSky | ≥ 1.0 | 长期 |
| CHDBits | ≥ 0.5 | 长期 |
| TTG | 免考核 | - |

## 3. 如何维护良好的分享率

### 3.1 抓取热门资源（大法）

**原理：**
热门资源下载的人多 → 上传机会多 → 分享率快速增长

**识别热门资源：**

在站点页面看：
- 🔥 热门标签
- 📥 下载人数多
- 📤 做种人数少

**时机：**
- 新资源发布的前 24 小时
- 周末、节假日
- 大片上映后

### 3.2 保持长时间做种

**设置持续做种：**

**qB → 设置 → 下载 → 种子管理**

| 选项 | 设置 |
|------|------|
| 做种比率限制 | 0（不限制） |
| 做种时间限制 | 0（不限制） |

**建议：**
- 保持 24/7 做种
- 使用盒子/NAS 持续运行
- 避免频繁开关 qB

### 3.3 选择合适资源

**优先选择：**

| 类型 | 原因 |
|------|------|
| 4K/蓝光 | 文件大，容易积累上传量 |
| 季番合集 | 长期有人下载 |
| 冷门精品 | 独家资源，容易获得魔力值 |

### 3.4 魔力值（Bonus）

**什么是魔力值？**
站点积分，可兑换上传量或邀请码

**获得方式：**
- 长时间做种
- 发布资源
- 帮助他人下载

**用途：**
- 兑换上传量（救急用）
- 购买邀请码
- 兑换 VIP 状态

3. 关键词过滤

## 6. 进阶技巧

### 6.1 合并下载

对于多个小文件：
- 使用 WinRAR 合并再下载
- 提高单次下载量
- 更容易积累上传

### 6.2 多客户端做种

**注意：⚠️ 违规行为**
同一个种子不要用多个客户端做种
会被检测为作弊

### 6.3 盒子使用

**为什么用盒子？**
- 24/7 持续做种
- 宽带更好（上行带宽高）
- 电费更低

**常见盒子：**
- 搬瓦工
- Vultr
- Google Cloud

## 8. 常见问题

### Q1: 分享率太低怎么办？

**紧急措施：**

1. **抓取热门资源**
   - 关注首页热门
   - 关注新资源发布

2. **使用魔力值兑换**
   - 避免完全封号

3. **上传原创资源**
   - 独家资源容易获得上传

### Q2: 下载很慢？

**可能原因：**

| 原因 | 解决 |
|------|------|
| 做种人数少 | 等待或换资源 |
| 分享率低 | 先做种攒分享率 |
| 网络问题 | 检查代理/网络 |
| 客户端设置 | 调整连接数/缓存 |

### Q3: 被站点封号了怎么办？

**申诉要点：**

1. 承认错误（如果确实违规）
2. 说明改正措施
3. 承诺以后遵守规则
4. 态度诚恳

**注意：** 很多站点封号是永久的

### Q4: 邀请码怎么获得？

**获得途径：**

- 长期活跃的老用户邀请
- 魔力值购买
- 参与站点活动
- 论坛/社区求助

> [!warning]
> **严禁买卖邀请码！**
> 一旦发现，买卖双方都会被封

## 9. 安全建议

### 9.1 隐私保护

**qB 设置：**

- 关闭 DHT（PT站点必须）
- 关闭 Local Peer Discovery
- 关闭 Peer Exchange（PT站点）

### 9.2 VPN 使用

**场景：**
- 访问被墙站点
- 提高连接质量

**注意：**
- ⚠️ 不要让 qB 全走 VPN
- 只用 VPN 访问站点页面
- qB 下载直连（站点要求）

### 9.3 定期备份

**备份内容：**

- 种子文件（.torrent）
- qB 配置
- 站点账号信息

**备份频率：**
- 重要配置：每月
- 种子文件：每周

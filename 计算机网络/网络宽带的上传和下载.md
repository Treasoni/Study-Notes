#网络 

先看[带宽和宽带](带宽和宽带.md)对这一方面有了解。
**先给结论（你先有个“正确认知”）**

> **家用宽带 = 重度偏向“下载”，严重限制“上传”**

所以你会遇到：

>- 上传文件慢 ❌
>- NAS 外网访问慢 ❌
>- 自建服务、推流、远程桌面卡 ❌

不是你设备差，**是宽带设计就是这样**
# 1. 为什么下载快、上传慢？（核心原因）

## 1.1 宽带本来是给「看内容的人」用的

运营商假设你主要在：

>- 刷视频
>- 看直播
>- 下载游戏
>- 网页浏览

这些行为的共同点是：

> **你在“收数据”，不是“发数据”**

所以：

- 下载给得多
- 上传给得少  
    （省成本，还符合大多数用户习惯）
    

## 1.2 技术层面的原因（不讲太深）

家用宽带大多是：

>- **PON / 光纤共享**
>- 一根光纤 → 分给一片小区

👉 上传如果不限制，会发生：

>- 大量用户同时“往外发”
>- 上行链路直接被打满
>- 整个小区都卡

所以运营商干脆：

> **强行限制每户的上传带宽**

# 2. 为什么局域网很快，外网却很慢？

这个反差很多人懵过 😄

## 2.1 局域网（家里）

>- 千兆网口
>- 实际：100–110 MB/s

### 外网（互联网）

>- 被宽带限制
>- 上传：2.5 MB/s

👉 差 **40 倍以上**

不是你配置错  
是**物理层级不一样**
